📌 Overview

This repository contains a structured collection of Java Programming assignments developed as part of my academic coursework.
All programs are implemented using core Java concepts, focusing on clean logic building, object-oriented programming, and real-world problem-solving approaches.

Each assignment is written with the goal of improving coding efficiency, debugging skills, and practical Java development knowledge.

The primary objectives of this repository are:

To gain hands-on experience with Object-Oriented Programming (OOP)

Step 2: Move into the Directory
cd JAVA_Assignment
Step 3: Compile the Java Program
javac FileName.java
Step 4: Execute the Program
java FileName

GUI programs require a system with Java Swing support.
Networking programs should be executed in two terminals:
Start Server first
Then run Client
